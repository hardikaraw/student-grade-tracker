# Student Grade Tracker

## 📌 Description
A console-based Java application that helps manage student grades.

## ✨ Features
- Enter student names
- Enter student marks
- Calculate average marks
- Find highest marks
- Find lowest marks
- Assign grades (A, B, C, D, F)
- Display a summary report

## 🛠 Technologies Used
- Java
- Scanner
- Arrays
- IntelliJ IDEA

## 🚀 How to Run
1. Open the project in IntelliJ IDEA.
2. Run `StudentGradeTracker.java`.
3. Enter the number of students.
4. Enter each student's name and marks.
5. View the generated report.
